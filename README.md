# Learning code

Some personal, small projects so I learn how to code. no AI slop.

## Projects

1. **Golang - CLI TODO List**: a simple Go app built with sqlite, bubble tea and lipgloss.
- Built using only documentation and a ([video reference](https://www.youtube.com/watch?v=_gzypL-Qv-g&list=LL&index=1&t=602s)).
- The app allows to create, edit, remove, and select tasks using keyboard inputs dynamically.
- Task persist to storage with [go-sqlite3](https://github.com/mattn/go-sqlite3).
- Simple visual interface built with [bubble tea](https://github.com/charmbracelet/bubbletea) and [tlipglosst](https://github.com/charmbracelet/lipgloss) TUI framework, (based on the [Elm architecture](https://guide.elm-lang.org/architecture/)).
